==========================================================================
ПРОЕКТ: "САЙТ-ПОРТФОЛИО IT-СПЕЦИАЛИСТА"
==========================================================================

ЦЕЛЬ ЗАДАНИЯ: 
Создать современную одностраничную визитку (Landing Page), объединив знания 
семантики HTML5, CSS Grid, Flexbox, анимаций и адаптивности.

--------------------------------------------------------------------------
1. СТРУКТУРА СТРАНИЦЫ (HTML)
--------------------------------------------------------------------------
* Header: 
  - Логотип и навигация (ссылки-якоря: #about, #skills, #projects, #contact).
  - Свойство position: sticky (меню всегда сверху при скролле).

* Hero Section (#about): 
  - Приветствие (h1), короткое описание и кнопка "Связаться".
  - Фото-аватар (используй object-fit: cover и border-radius: 50%).

* Skills Section (#skills): 
  - Заголовок h2.
  - Сетка навыков (минимум 4 карточки: HTML, CSS, JS, Git).
  - Использовать CSS Grid.

* Projects Section (#projects): 
  - Минимум 3 карточки проектов.
  - Каждая карточка: картинка-заглушка, заголовок h3, описание, ссылка.

* Contact Section (#contact): 
  - Форма обратной связи: имя (input), email (input), сообщение (textarea).
  - Кнопка отправки (button).

* Footer: 
  - Копирайт и ссылки на соцсети (иконки или текст).

--------------------------------------------------------------------------
2. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ (CSS)
--------------------------------------------------------------------------
1. ПЕРЕМЕННЫЕ: Объяви в :root основные цвета проекта и шрифты.
2. RESET: Обнули margin и padding для всех элементов (* {box-sizing: border-box}).
3. FLEXBOX: Используй для выравнивания элементов в Header и Hero-секции.
4. GRID: Для секции Skills используй: grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)).
5. ЭФФЕКТЫ:
   - Плавные переходы (transition: 0.3s) для всех ссылок и кнопок.
   - Эффект :hover для карточек проектов (увеличение или появление тени).
   - Псевдоэлемент ::after для создания декоративной линии под заголовками h2.
6. АДАПТИВНОСТЬ:
   - При ширине экрана < 768px: сетка проектов и навыков перестраивается в 1 колонку.
   - Текст заголовков уменьшается.

--------------------------------------------------------------------------
3. ДОПОЛНИТЕЛЬНЫЕ ЗАДАНИЯ (PRO)
--------------------------------------------------------------------------
- Стилизуй скроллбар браузера (::-webkit-scrollbar) под цвета сайта.
- Добавь поддержку темной темы через @media (prefers-color-scheme: dark).
- В комментариях HTML напиши Emmet-команду, которой ты сгенерировал 
  секцию с карточками проектов.

--------------------------------------------------------------------------
С чего начать:
1. Создай index.html и style.css.
2. Пропиши базовые стили (шрифты, цвета).
3. Верстай блоки по очереди, проверяя отображение в браузере.
